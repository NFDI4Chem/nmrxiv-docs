# Contributing to nmrXiv

## Contribution guidelines

### How to contribute to nmrXiv

#### Did you find a bug?

* **Do not open up a GitHub discussion issue if the bug is a security vulnerability in nmrXiv**, and instead contact us here: <i>info@nmrxiv.org</i> (This email is not ready yet).

* **Ensure the bug was not already reported** by searching on GitHub under [nmrXiv discussions issues](https://github.com/NFDI4Chem/nmrxiv/discussions/categories/issues).

* If you're unable to find an open issue addressing the problem, [open a new discussion](https://github.com/NFDI4Chem/nmrxiv/discussions/new), and <b>Select category</b> as <b>Issues</b>. Be sure to include a <b>Title</b> and clear <b>description</b>, with as much relevant information as possible, and a **code sample** or an **executable test case** demonstrating the expected behavior that is not occurring.
  
#### **Did you write a patch that fixes a bug?**

* Open a new GitHub pull request with the patch.

* Ensure that the pull request description clearly describes the problem and solution. Include the relevant issue number if applicable.

* Before submitting, please read the <i>Contributing to nmrXiv</i> guide to know more about coding conventions and benchmarks.
